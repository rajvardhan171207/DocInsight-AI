# 📄 DocInsight AI

An AI-powered Document Intelligence System built using Flask and Google's Gemini AI.

DocInsight AI allows users to upload PDF, DOCX, and TXT documents, automatically analyze their contents, generate AI summaries, classify document types, extract keywords, visualize document statistics, and chat with the document.

---

## 🚀 Features

- 📄 Upload PDF, DOCX, and TXT files
- 🤖 AI Document Summary (Gemini AI)
- 💬 Ask Questions About the Document
- 🏷️ Automatic Document Classification
- 😊 AI Sentiment Analysis
- 📊 Document Statistics
- ☁️ Word Cloud Generation
- 📈 Top 10 Word Frequency Chart
- 🔑 Keyword Extraction
- 📥 PDF Report Generation
- 📋 Copy Summary
- ⏳ Loading Animation
- 📱 Responsive User Interface
